# Workflow: Portfolio 2.0 Development

## Local Development
- Run `npm install` for new dependencies.
- Use `npm run dev` for hot-reloading development.
- Test responsive breakpoints (Desktop, Tablet, Mobile) in DevTools.

## Review Process
- Visual review for "Midnight Nebula" aesthetic consistency.
- Performance check (Lighthouse) for backdrop-blur impact.
- Accessibility check (Color contrast, Aria labels).

## Deployment
- Push to main/master to trigger GitHub Actions (as configured).
